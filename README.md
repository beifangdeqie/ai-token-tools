## 💰 成本优化技巧

1. **选对模型**：简单任务用便宜模型，复杂任务用强模型
2. **缓存结果**：相同问题不重复调用
3. **批量请求**：减少HTTP开销
4. **Token压缩**：精简prompt，减少输入token
